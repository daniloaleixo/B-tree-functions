# Funções para B-trees

## Introdução
Projeto realizado para a disciplina de Estrutura de Dados para estudar B-trees

## Enunciado
Neste exercício‐programa, a tarefa consiste em implementar funções relativas as árvores B. Seu EP deve ler do teclado, inicialmente, a ordem d da árvore B e posteriormente, se o usuário deseja (i) buscar, (ii) inserir, (iii) remover, uma chave, ou (iv) finalizar o processo. Para cada caso de (i) a (iii), seu EP deve imprimir de forma “razoável”, a estrutura da árvore B. 

## Executar ##
Para executar basta compilar o programa com o Makefile
```sh
make
```
E depois executar o programa para verificação de conectividade 
  ```sh
  ep4
  ```
